# IT-351-Project
operates two campuses: a Main Campus and a Branch Campus. Each campus network is treated as a separate WAN site. Your task is to design and configure, in Cisco Packet Tracer, a network that successfully transmits data packets from  device in any LAN of WAN 1 to a device in any LAN of WAN 2. Each WAN contains 2 LANs, and each LAN has 3 end devices.
